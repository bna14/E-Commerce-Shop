Sales Service

app.display_available_goods()[source]

Display available goods with their names and prices.

Fetches goods from the Inventory Service and filters their details to include only name, price, and ID.

Returns:

A JSON list of goods with id, name, and price. status_code (int): HTTP status code, 200 for success or an error code.

Return type:

list

app.get_goods_details(item_id)[source]

Get detailed information about a specific good.

URL parameter: - item_id: ID of the item

Returns: - 200: Item details - 404: Item not found - 500: Unable to fetch item details - 503: Inventory Service is not available

app.get_purchase_history(username)[source]

Get the purchase history for a specific customer.

URL parameter: - username: Username of the customer

Returns: - 200: List of purchase history - 404: No purchase history found for the user

app.index()[source]

Health check endpoint.

Returns: - 200: Service is running

app.process_sale()[source]

Process a sale when a customer purchases a good.

Request JSON should contain: - username: Username of the customer - item_id: ID of the item being purchased - quantity: Quantity of the item being purchased (default is 1)

Returns: - 200: Sale processed successfully - 400: Missing required fields, insufficient stock, or insufficient balance - 404: Item or customer not found - 500: Failed to deduct balance or stock - 503: Inventory Service is not available
